Another Map Generator

Languages:

Java

Summary:

A generator that creates 3d maps (using layers) for a 4X game

Details:

An interpreter that reads in config files and generates a map for them, including customizability so if players wanted to mod the maps, they could extend the map and even add or remove layers of the map. It is written abstractly so that the GUI (currently only a simple terminal view) can easily be modified, and with only a few changes, the scripts could work with hex maps
